Demultiplexer Output

R-S Flip-Flop Next State

Recall the R-S flip-flop truth table: R=0,S=0 holds the current state, R=0,S=1 sets Q to 1, R=1,S=0 resets Q to 0, and R=1,S=1 is disallowed. Given the current state and inputs, give the next Q.
